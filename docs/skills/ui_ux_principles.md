# 🎨 SKILL: PRINCÍPIOS DE UI & UX (VISUAL EXCELLENCE)

## 🎯 Objetivo
Garantir que a plataforma não seja apenas funcional, mas visualmente deslumbrante, intuitiva e proporcione uma experiência premium ("WOW factor").

## 💎 Princípios de Design Visual (UI)
1. **Estética Premium:** Evite cores genéricas. Use paletas harmoniosas, modos escuros elegantes e gradientes sutis.
2. **Tipografia Moderna:** Use fontes legíveis e com hierarquia clara (ex: Inter, Outfit). Diferencie pesos para guiar o olhar do usuário.
3. **Micro-interações:** Adicione feedbacks visuais sutis (hovers, transições suaves, estados de carregamento elegantes) para fazer a interface parecer "viva".
4. **Espaçamento e Respiro:** Respeite o espaço em branco (white space). Não amontoe informações; deixe o conteúdo respirar para reduzir a carga cognitiva.
5. **Consistência Atômica:** Todos os botões, cards e inputs devem seguir o mesmo padrão visual definido no Design System.

## 🚀 Experiência do Usuário (UX)
1. **Foco na Tarefa:** Minimize o número de cliques para completar ações críticas (ex: criar uma aula, assistir um vídeo).
2. **Feedback Imediato:** O usuário nunca deve se perguntar "será que funcionou?". Use toasts, progress bars e esqueletos de carregamento (skeletons).
3. **Acessibilidade (a11y):** Contraste de cores adequado, tamanhos de fonte legíveis e suporte a navegação por teclado/leitores de tela.
4. **Mobile First:** A experiência no app deve ser tão fluida quanto no desktop, adaptando gestos e tamanhos de toque (mínimo 44x44px).
5. **Prevenção de Erros:** Use validação em tempo real e mensagens de erro amigáveis que ajudem o usuário a resolver o problema.

## 🧱 Elementos do CMS
- **Canvas Visual:** A interface de edição deve ser "What You See Is What You Get" (WYSIWYG), aproximando o editor do resultado final do aluno.
- **Hierarquia de Blocos:** Use sombras e bordas sutis para separar visualmente os blocos de conteúdo no player.
